# 📈 股價資料分析與視覺化 - 台積電 (2330.TW)

### 📅 日期：2025/04/21  
### 🎯 類型：個人學習 / 交易助理技能養成計畫 Day 1

---

## 專案簡介

本專案為轉職量化交易助理的第一步，使用 Python 抓取台灣股市資料，並視覺化股價趨勢圖，奠定日後技術分析、策略開發與金融理解的基礎。

---

## 技術工具與環境

- Python + Jupyter Notebook
- yfinance (抓取歷史股價資料)
- matplotlib (畫圖)
- 作業系統：Windows 10

---

## 分析內容

- 資料來源：台積電（2330.TW）過去一個月歷史股價
- 欄位重點：Open、High、Low、Close、Volume
- 可視化圖表：收盤價趨勢線
